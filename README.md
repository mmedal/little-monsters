## Little Monsters

Brightwheel's little monster avatar generator.

### System Dependencies:
* gd
```
brew install gd
```

### Installation
```
nvm use
yarn install
yarn develop
```

Navigate to `127.0.0.1:3000/api/avatars/:identifier` where `:identifier` is
the string you would like to use to generate your avatar.
